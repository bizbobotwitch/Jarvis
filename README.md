⚡ J.A.R.V.I.S. Desktop Assistant & Code Workbench (v1.0)A simple Python project that runs a voice-activated desktop assistant on your computer to help you manage your system and work with local code files.
🛠️ What This Project DoesThis system is focused entirely on providing a local, interactive AI companion on your machine:1. Voice-Activated Desktop Assistant (Mark1.py)A desktop application with a visual interface that lets you talk to an AI assistant using your microphone or by typing messages.Voice Commands: Ask the assistant to open local apps (like Calculator or Notepad), search the web, tell you the time, check what games are running on your computer, or take screenshots.Code Workbench: Load code files directly into the window, ask the assistant to check them for errors or recommend updates, and save your edited code back to your computer.📂 Project StructureMark1.py — The primary voice-activated desktop assistant and code workspace script.jarvis_config.json — The local configuration file storing your active settings.Jarvis_Documents/ — A safe folder created on your computer for reading and writing files.🚀 Quick Start GuideFollow these steps to download, install, and run J.A.R.V.I.S. on your system.Step 1: Clone the RepositoryClone the project files directly to your machine. Once finished, the files will be located right inside your active user folder:git clone [https://github.com/bizbobotwitch/Jarvis](https://github.com/bizbobotwitch/Jarvis)
Online repository link: https://github.com/bizbobotwitch/Jarvis/tree/mainStep 2: Install Python & DependenciesMake sure you have Python 3.8 or newer installed. Open your terminal or Command Prompt and run the following command to install the required libraries:pip install SpeechRecognition pyttsx3 Pillow google-generativeai
Step 3: Start the AssistantGet a free Gemini API key from Google AI Studio.Run the assistant script from your terminal:python Mark1.py
Once the window opens, click on the Settings / System Matrices tab.Paste your Gemini API key into the Primary API Key field and click Apply & Save Settings.Select your microphone from the dropdown list on the main dashboard and click INITIALIZE CORE to start talking to Jarvis!
