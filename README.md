⚡ J.A.R.V.I.S. Desktop Assistant & Code Workbench (v1.0)A simple Python project that runs a voice-activated desktop assistant on your computer to help you manage your system and work with local code files.🛠️ What This Project DoesThis system is focused entirely on providing a local, interactive AI companion on your machine:1. Voice-Activated Desktop Assistant (local_voice_agent.py)A desktop application with a visual interface that lets you talk to an AI assistant using your microphone or by typing messages.Voice Commands: Ask the assistant to open local apps (like Calculator or Notepad), search the web, tell you the time, check what games are running on your computer, or take screenshots.Code Workbench: Load code files directly into the window, ask the assistant to check them for errors or recommend updates, and save your edited code back to your computer.📂 Project Structure├── local_voice_agent.py   # Voice-activated desktop assistant and code workspace
├── jarvis_config.json     # Configuration file storing your settings
└── Jarvis_Documents/      # Safe folder on your computer for reading and writing files
🚀 Full Startup GuideFollow these steps to download, install, and run J.A.R.V.I.S. on your computer.Step 1: Install PythonJ.A.R.V.I.S. requires Python to run. If you don't have it installed yet:Download Python 3.8 or newer from the Official Python Website.Important: During installation on Windows, make sure to check the box that says "Add Python to PATH" before clicking install.Open your terminal (Command Prompt on Windows, Terminal on macOS) and verify the installation by typing:python --version
Step 2: Clone the RepositoryOpen your terminal and clone the repository. The files will be downloaded directly into your active user folder:git clone https://github.com/bizbobotwitch/Jarvis
After cloning, navigate into the project directory:cd Jarvis
Step 3: Install Required DependenciesInstall the Python libraries required for speech recognition, audio playback, screenshot analysis, and AI connection:pip install SpeechRecognition pyttsx3 Pillow google-generativeai
Note for macOS/Linux Users: If you run into any installation errors with SpeechRecognition or audio tools, you may need to install audio system packages first:macOS: Install Homebrew and run brew install portaudioLinux: Run sudo apt-get install portaudio19-dev python3-pyaudioStep 4: Get a Free Google Gemini API KeyTo allow J.A.R.V.I.S. to think and respond, you need an API key:Go to Google AI Studio.Sign in with any Google account.Click on "Get API Key" and create a new key.Copy the key to your clipboard.Step 5: Start and Configure J.A.R.V.I.S.Run the script from your terminal:python Mark1.py
Once the window opens, click on the Settings / System Matrices tab.Paste your Gemini API key into the Primary API Key field.Customize your name and your assistant's name if you want, and click Apply & Save Settings.Go back to the main console tab, select your microphone from the dropdown list, and click INITIALIZE CORE / BOOT UP AI CORE to start speaking with Jarvis!
